# 🚀 Quick Start Guide

## For Local Testing

### Option 1: Direct Browser Opening
1. Download or clone this repository
2. Open `index.html` directly in your web browser
3. Start learning!

### Option 2: Using Python (Recommended)
```bash
# Navigate to the project directory
cd daily-routine-lesson

# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Then open `http://localhost:8000` in your browser.

### Option 3: Using Node.js
```bash
# Install http-server globally (one-time)
npm install -g http-server

# Run from project directory
http-server
```
Then open the provided URL in your browser.

## For GitHub Pages Hosting

### Step 1: Create GitHub Repository
1. Go to [github.com](https://github.com)
2. Click "New repository"
3. Name it `daily-routine-lesson`
4. Click "Create repository"

### Step 2: Upload Files
```bash
# In your project directory
git init
git add .
git commit -m "Initial commit: Daily Routine Lesson"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/daily-routine-lesson.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings"
3. Scroll to "GitHub Pages"
4. Select "main" branch
5. Click "Save"

### Step 4: Access Your Site
Your site will be live at:
```
https://YOUR_USERNAME.github.io/daily-routine-lesson/
```

## Features Overview

### 📚 Learning Section
- 12 interactive slides covering daily routine vocabulary and grammar
- Navigate with Previous/Next buttons or arrow keys
- Click slide indicators to jump to specific slides

### 🎯 Activities Section
1. **Matching Exercise** - Drag times to activities
2. **Fill in the Blanks** - Complete sentences with correct words
3. **Multiple Choice** - Select correct grammar answers
4. **Adverb Placement** - Choose correct adverbs
5. **Sequence Ordering** - Arrange morning routine in order
6. **Free Writing** - Describe your ideal day with feedback

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Slides not showing | Ensure JavaScript is enabled; try refreshing |
| Activities not responding | Clear browser cache; check console for errors (F12) |
| Mobile display issues | Zoom to 100%; try different browser |
| Drag-drop not working | Use Chrome/Firefox; ensure JavaScript enabled |
