# 📚 Mastering Your Daily Routine - Interactive English Lesson

An engaging, interactive web-based English learning platform focused on teaching daily routine vocabulary, grammar, and expressions. Perfect for ESL/EFL learners of all levels!

## 🌟 Features

### Learning Section
- **12 Interactive Slides** with comprehensive content covering:
  - Why talk about routines
  - Morning, afternoon, evening, and night vocabulary
  - Time expressions (at, in the)
  - Present Simple grammar (all persons)
  - Adverbs of frequency
  - Sequencing words
  - Summary and key takeaways

- **Slide Navigation**
  - Previous/Next buttons for easy navigation
  - Keyboard support (arrow keys)
  - Slide indicators with direct access
  - Slide counter showing progress

### Interactive Activities
Six engaging activities with instant feedback:

1. **Match the Time with Activity** - Drag-and-drop matching exercise
2. **Complete the Sentences** - Fill-in-the-blank vocabulary practice
3. **Choose the Correct Answer** - Multiple-choice grammar questions
4. **Place the Adverb Correctly** - Adverb placement practice
5. **Order the Sequence** - Arrange morning routine activities in correct order
6. **Describe Your Ideal Day** - Free writing with AI-powered feedback analysis

### User Experience
- ✅ **Instant Feedback** - Immediate results for all activities
- 🎨 **Beautiful Design** - Modern, colorful, and engaging interface
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile
- ⌨️ **Keyboard Support** - Navigate slides with arrow keys
- 🎯 **Progress Tracking** - See your score and performance
- 🌈 **Smooth Animations** - Delightful transitions and interactions

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or installation required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/daily-routine-lesson.git
   cd daily-routine-lesson
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Python 2
     python -m SimpleHTTPServer 8000
     
     # Node.js (if you have http-server installed)
     http-server
     ```

3. **Access the site**
   - Open `http://localhost:8000` in your browser

## 📖 How to Use

### Learning Section
1. Click the **"Learning"** tab in the navigation
2. Use the **Previous (❮)** and **Next (❯)** buttons to navigate slides
3. Or click on the **slide indicators** at the bottom to jump to a specific slide
4. Use **arrow keys** on your keyboard for quick navigation
5. Review all 12 slides to understand daily routine vocabulary and grammar

### Activities Section
1. Click the **"Activities"** tab to access interactive exercises
2. Complete each activity by:
   - **Matching**: Drag items to their correct pairs
   - **Fill-in-the-Blank**: Type the correct word in each blank
   - **Multiple Choice**: Select the correct answer
   - **Adverbs**: Click to select the correct adverb
   - **Sequencing**: Drag items to arrange them in order
   - **Description**: Write your ideal day description
3. Click **"Check Answers"** or **"Check Order"** to see instant feedback
4. Review the feedback and try again if needed

- **Morning Routine**: Wake up, get up, brush teeth, wash face, have breakfast, get dressed, leave house

- **Night Routine**: Have a shower, read a book, set the alarm, go to bed
- **Household Chores**: Do laundry, sweep floor, do dishes, take out trash
- **Work/Study**: Sign in, attend meetings, check emails, meet deadlines

### Grammar Topics
- Present Simple (all persons)
- Third-person singular "s" rule
- Time expressions (at, in the)
- Adverbs of frequency
- Sequencing words

## 🐛 Troubleshooting

### Slides not displaying
- Ensure JavaScript is enabled in your browser
- Check browser console for errors (F12 → Console)
- Try refreshing the page

### Activities not working
- Clear browser cache and reload
- Check that all files (HTML, CSS, JS) are in the same directory
- Verify JavaScript is enabled

### Mobile display issues
- Check viewport meta tag in HTML
- Test in different mobile browsers
- Zoom to 100% for best experience
